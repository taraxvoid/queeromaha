# Security Policy

## Vulnerabilities

For "nice to haves" or low CVSS vulns, please make a PR.

For high severity issues, see the GitHub Security Advisory feature

## Malicious NPM Packages

There is a 7-day delay on package updates, set in `bunfig.toml` to mitigate some risk. Please raise an issue / PR if a package is potentially compromised.

## Compromise / Exploitation

Ping Tara (or any other admins) on Signal or Protonmail.

## Privacy

For data removal requests email privacy@queeromaha.net

No cookies or marketing analytics are used

If you need information wiped from the git history of this repo please reach out.

## Abuse / Bad Behavior

Ping Tara on Signal or Protonmail for sensitive issues

Site admins are NOT able to intervene in personal issues or safety problems at venues/spaces/groups/individuals
