# Security Policy

## Vulns

For "nice to haves" or low CVSS vulns, make a PR. 

For high severity issues, see the GitHub Security Advisory feature

For active exploitation / compromise including any issues with the live hosting of the site ping Tara (or any other admins) on Signal or Protonmail. 

## Privacy

For data removal requests email privacy@queeromaha.net

No cookies or gross marketing analytics are used

If you need information wiped from the git history of this repo please reach out.

## Abuse / Bad Behavior

Ping Tara on Signal or Protonmail for sensitive issues such as if a group / space is taken over by TERFs etc.

Site admins are NOT empowered to intervene in personal issues or safety problems at venues or spaces and will NOT carry water for pet issues or engage in gossip.
